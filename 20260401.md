### 🖥️ Informações do Sistema

* **Como verificar o nome do computador:**
    Para ver o nome da máquina (hostname), basta digitar:
    ```bash
    hostname
    ```
    *(Dica: o comando `hostnamectl` mostra o nome e várias outras informações úteis).*

* **Como verificar a versão do sistema operacional:**
    Para saber qual versão do Ubuntu está rodando:
    ```bash
    lsb_release -a
    ```
    *(Alternativa: `cat /etc/os-release`)*

---

### ⚙️ Hardware (Processamento e Memória)

* **Como verificar a memória RAM do computador:**
    Para ver o total de RAM, o quanto está em uso e o quanto está livre (o `-h` deixa os números mais fáceis de ler, em Megabytes e Gigabytes):
    ```bash
    free -h
    ```

* **Como verificar a CPU do computador:**
    Para listar os detalhes do processador (núcleos, modelo, velocidade):
    ```bash
    lscpu
    ```

### 💾 Discos e Armazenamento

* **Como verificar o disco do computador:**
    Para ver as partições e o espaço livre/usado no disco:
    ```bash
    df -h
    ```
    Para listar todos os discos e partições físicas conectadas:
    ```bash
    lsblk
    ```

* **Como montar um pendrive (Somente Root):**
    1. Primeiro, descubra qual é o nome do dispositivo do pendrive (ex: `sdb1`) conectando-o e rodando: `lsblk` ou `sudo fdisk -l`.
    2. Crie uma pasta para "ancorar" o pendrive:
       ```bash
       sudo mkdir /mnt/pendrive
       ```
    3. Monte o pendrive nessa pasta (substitua `sdb1` pelo nome correto do seu pendrive):
       ```bash
       sudo mount /dev/sdb1 /mnt/pendrive
       ```

### 🌐 Redes

* **Como verificar a rede (IP, Gateway e DNS):**
    * **IP:** `ip a` ou `ip addr show`
    * **Gateway (Roteador):** `ip route` (procure a linha que começa com "default via")
    * **DNS:** `resolvectl status`

* **Como mudar a rede (IP, Gateway e DNS):**
    No Ubuntu moderno, a configuração de rede pelo terminal é feita através de uma ferramenta chamada **Netplan**.
    1. Você precisa editar o arquivo de configuração (geralmente em `/etc/netplan/`). Exige root:
       ```bash
       sudo nano /etc/netplan/01-network-manager-all.yaml
       ```
    2. Dentro do arquivo, você define o IP estático, gateway e DNS seguindo a estrutura YAML.
    3. Depois de salvar, aplique as mudanças:
       ```bash
       sudo netplan apply
       ```
    *(Nota: Se estiver usando o Ubuntu com interface gráfica, é muito mais fácil alterar isso acessando as **Configurações > Rede** e editando a conexão diretamente).*

### 📦 Gerenciamento de Pacotes (Programas)

* **Como listar os pacotes instalados:**
    Para ver tudo o que está instalado no sistema:
    ```bash
    apt list --installed
    ```

* **Como procurar um pacote:**
    Se você quer instalar um programa (ex: `htop`), mas quer ver se ele existe no repositório primeiro:
    ```bash
    apt search nome-do-pacote
    ```

* **Como instalar um pacote (Somente Root / Instrutor):**
    Para instalar um novo programa, você usa o gerenciador `apt` com privilégios de administrador:
    ```bash
    sudo apt install nome-do-pacote
    ```
