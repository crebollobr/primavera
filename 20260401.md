
Bem-vindos! Se esta é a sua primeira vez lidando com o Linux, respire fundo: o terminal não morde. Hoje, vamos desmistificar como o Linux organiza suas pastas, como nos movemos por elas e por que a forma como ele guarda suas configurações é um dos seus maiores "superpoderes".

---

## 1. O GPS do Linux: Onde estou e para onde vou?

No Windows, você está acostumado a clicar em pastas com o mouse. No terminal do Linux, fazemos a mesma coisa, mas usando três comandos mágicos: `pwd`, `ls` e `cd`.

### `pwd` (Print Working Directory) - "Onde eu estou?"
Imagine que você acordou de olhos vendados em uma sala. O comando `pwd` tira a sua venda e diz exatamente em qual pasta você está no momento.
```bash
pwd
```
*Exemplo de resposta:* `/home/aluno`

### `ls` (List) - "O que tem aqui?"
Agora que você sabe onde está, quer saber o que tem ao seu redor. O comando `ls` lista todos os arquivos e pastas que estão dentro do diretório atual.
```bash
ls
```
*Dica:* Se você usar `ls -l`, ele mostra uma lista detalhada, como uma tabela!

### `cd` (Change Directory) - "Como eu vou para outro lugar?"
Você viu uma pasta chamada `Documentos` e quer entrar nela. O comando `cd` é o equivalente a dar um "duplo clique" em uma pasta.
```bash
cd Documentos
```
Para voltar uma pasta para trás (sair da sala), você usa dois pontos:
```bash
cd ..
```

## 2. A Árvore de Diretórios e a Famosa Pasta `/etc`

No Linux, não existem "Disco C:" ou "Disco D:". Tudo começa na **Raiz**, representada por uma simples barra: `/`. Todas as pastas do sistema crescem a partir dessa raiz, como os galhos de uma árvore.

### O coração das configurações: `/etc`
Se você navegar até a pasta `/etc` (usando o comando `cd /etc`), você entrará no "painel de controle" do Linux. 
É nessa pasta que o sistema operacional e os programas guardam todas as suas configurações. Quer mudar como a rede funciona? Está no `/etc`. Quer configurar um servidor web? Está no `/etc`.

---

## 3. O Superpoder do Linux: "Tudo é Arquivo de Texto"

Aqui entra um dos conceitos mais bonitos e poderosos do Linux: **as configurações são apenas arquivos de texto puro.**

No Windows, as configurações do sistema ficam escondidas em um "Registro" complexo, que precisa de programas especiais para ser lido ou alterado. No Linux, se você quiser mudar uma configuração, você só precisa abrir o arquivo de texto, mudar a palavra, e salvar.

### Quais as vantagens disso?
* **Simplicidade:** Você pode abrir e editar configurações com qualquer editor de texto simples.
* **Fácil de fazer backup:** Para salvar suas configurações, basta copiar a pasta `/etc` para um pendrive.
* **Transparência:** Você pode ler exatamente o que o sistema está fazendo, com seus próprios olhos. Não há código secreto.
* **Automação:** É muito fácil criar scripts (robozinhos) para alterar textos, o que facilita configurar dezenas de computadores de uma vez só.

## 4. Preparação para o Laboratório Prático

Para o nosso laboratório de hoje, usaremos os três comandos que aprendemos (`pwd`, `ls`, `cd`). Mas, como falamos que as configurações são arquivos de texto, seria legal *ler* um deles, certo?

Para isso, vou apresentar um comando extra muito simples, exclusivo para o nosso laboratório:
* **`cat`**: Este comando pega um arquivo de texto e "cospe" o conteúdo dele na sua tela. Ele serve apenas para leitura. Se você usar o `cat` em um arquivo, você não vai quebrar nada!

*Exemplo de uso:* `cat nome-do-arquivo.txt`

## 5. Laboratório Prático: Explorando o Sistema

Siga os passos abaixo no seu terminal. O objetivo é sairmos da nossa pasta pessoal, visitarmos a famosa pasta `/etc`, lermos uma configuração real do sistema operacional em formato de texto e voltarmos para casa em segurança.

**Passo 1: Descubra onde você está.**
Digite o comando abaixo e aperte Enter. Anote mentalmente qual é a sua "casa" (provavelmente algo como `/home/seu-usuario`).
```bash
pwd
```

**Passo 2: Viaje para o "painel de controle" do Linux.**
Vamos entrar na pasta de configurações.
```bash
cd /etc
```

**Passo 3: Confirme que você chegou.**
Use o GPS novamente para ter certeza de que você mudou de pasta. A resposta deve ser `/etc`.
```bash
pwd
```

**Passo 4: Olhe ao redor!**
Vamos listar tudo o que existe dentro do `/etc`. Prepare-se, a lista é grande, pois são as configurações do computador inteiro!
```bash
ls
```

**Passo 5: Leia um arquivo de configuração real.**
Dentro do `/etc`, existe um arquivo de texto chamado `os-release` que guarda as informações sobre a versão do seu Linux. Vamos usar o comando que aprendemos para ler este texto.
```bash
cat os-release

cat passwd
```
> *Observe a saída na tela. Viu como as configurações do sistema são literalmente linhas de texto simples que qualquer pessoa consegue ler?*

> 💡 **Tarefa:** tire um printscreen ou copie o texto e envie por e-mail.

**Passo 6: Volte para casa.**
Para voltar rapidamente para a sua pasta de usuário (sua "casa" no sistema), você pode digitar `cd` e simplesmente apertar Enter, sem escrever mais nada.
```bash
cd
```

**Passo 7: Verifique se você chegou em casa.**
```bash
pwd
```
